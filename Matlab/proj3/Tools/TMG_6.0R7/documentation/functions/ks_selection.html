KS_SELECTION - implements the heuristic method from [2] for 
  the selection of the number of factors from each cluster 
  used in the Clustered Latent Semantic Indexing method [1].
    N_ST=KS_SELECTION(A, N_COLS, ALPHA_VAL, L) returns in N_ST 
    a vector of integers denoting the number of factors (sum 
    equals L) selected from each cluster of the tdm A. N_COLS 
    is a vector containing the last column index for each column 
    block, while ALPHA_VAL is a value in [0, 1].

    Approximation Scheme from Clustered Term-Document Matrices.

    Classification", In Proc. of the 2006 IEEE International
